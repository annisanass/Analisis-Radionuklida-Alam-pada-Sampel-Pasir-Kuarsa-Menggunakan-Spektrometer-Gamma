# Natural Radionuclide Analysis using Gamma Spectrometry

## Overview
This project presents the analysis of natural radionuclide concentrations in quartz sand samples collected from former tin mining areas in Bangka Belitung, Indonesia. The study utilizes gamma spectrometry to identify and quantify radionuclides from uranium and thorium decay series.

## Objective
The main objective of this study is to evaluate the concentration levels of natural radionuclides in post-mining quartz sand and assess potential radiation exposure risks.

## Methodology
- Sample collection from former tin mining areas
- Gamma-ray spectrum acquisition using a gamma spectrometer
- Data analysis using *Y-Spect software* with IAEA nuclide library
- Calculation of radionuclide concentrations (Bq/kg)

## Radionuclides Analyzed
- Pb-214  
- Bi-214  
- K-40  
- Pb-212  
- Ac-228  
- Tl-208  

## Results
The average radionuclide concentrations obtained are:

| Radionuclide | Concentration (Bq/kg) |
|-------------|----------------------|
| Pb-214      | 404.43               |
| Bi-214      | 458.92               |
| K-40        | 44.94                |
| Pb-212      | 415.30               |
| Ac-228      | 931.94               |
| Tl-208      | 959.69               |

The results indicate relatively high concentrations of *Ac-228* and *Tl-208*, suggesting significant gamma radiation presence.

## Impact
High radionuclide concentrations may pose health risks, especially for mining workers exposed to long-term gamma radiation. This study highlights the importance of radiation monitoring in post-mining environments.
